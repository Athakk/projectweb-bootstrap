Project Yang Pernah Saya Kerjakan

Human Resource Management System

Salah satu project yang saya kerjakan selama prakerin adalah membuat Aplikasi Human Resource Management System (HRMS) berbasis web menggunakan Laravel 9, dibantu dengan Vue JS 3, dan base web milik perusahaan. Aplikasi ini digunakan untuk memanagement pegawai, dilengkapi dengan absensi serta pengelompokan gaji dan juga terdapat 2 user (sebagai admin dan sebagai user/pegawai).

Point Of Sale

Salah satu project yang saya kerjakan selama prakerin adalah membuat Aplikasi Point Of Sale(POS) Multi Perusahaan berbasis web menggunakan Laravel 9, dibantu dengan Vue JS 3, dan base web milik perusahaan. Aplikasi ini digunakan untuk memanagement penjualan(SO), pembelian(PO), Stok Barang yang juga bisa digunakan untuk mengatur keuangan perusahaan tersebut serta terdapat 2 user (sebagai admin dan sebagai user).
